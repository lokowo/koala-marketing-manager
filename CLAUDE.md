## рҹҡЁ жҢҮд»Өжү§иЎҢеҗҲзәҰпјҲжҜҸжқЎжҢҮд»Өеҝ…йЎ»йҒөе®Ҳпјү

жҜҸжқЎ Claude Code жҢҮд»ӨйғҪжҳҜдёҖд»ҪеҗҲзәҰгҖӮжү§иЎҢеүҚе…ҲжЈҖжҹҘиҝҷ 6 жқЎпјҡ

1. **й”ҡе®ҡеҸҜйӘҢиҜҒдәӢе®һ** вҖ” зЎ®и®ӨжҢҮд»ӨйҮҢзҡ„еүҚзҪ®зҠ¶жҖҒпјҲcommit hashгҖҒVercel зҠ¶жҖҒгҖҒеҪ“еүҚжҠҘй”ҷпјүгҖӮеҰӮжһңеүҚзҪ®дёҚжҲҗз«ӢпјҢе…ҲжҠҘе‘ҠеҶҚжү§иЎҢгҖӮ
2. **зәҝжҖ§дё»жӯҘйӘӨ + еұҖйғЁеҲҶж”Ҝ** вҖ” жҢү 1вҶ’2вҶ’3вҶ’4вҶ’5 йЎәеәҸжү§иЎҢгҖӮеҸӘеңЁзҺ°е®һжңүеӨҡз§Қжғ…еҶөж—¶иө°еҲҶж”ҜпјҢе…¶д»–зӣҙз»ҷгҖӮ
3. **жҜҸжӯҘйӘҢиҜҒй—ё** вҖ” жҜҸдёӘжӯҘйӘӨжңүжҳҺзЎ®зҡ„"е®ҢжҲҗ"е®ҡд№үпјҲbuild йҖҡиҝҮгҖҒcurl иҝ”еӣһ 200гҖҒecho иҫ“еҮә вң…пјүгҖӮйӘҢиҜҒдёҚиҝҮдёҚиғҪиҝӣе…ҘдёӢдёҖжӯҘгҖӮ
4. **зәҰжқҹжҳҜгҖҢдёҚи®ёгҖҚжё…еҚ•** вҖ” дёҘж јйҒөе®ҲжҢҮд»Өдёӯзҡ„зҰҒжӯўйЎ№гҖӮжІЎжңүжҸҗеҲ°зҡ„ж–Үд»¶дёҚи®ёж”№пјҢжІЎжңүиҰҒжұӮзҡ„еҠҹиғҪдёҚи®ёеҠ гҖӮ
5. **ејәеҲ¶жұҮжҠҘ** вҖ” е®ҢжҲҗеҗҺеҝ…йЎ»жҠҘе‘ҠпјҡйҮҮз”ЁдәҶе“Әз§Қдҝ®жі•гҖҒж”№дәҶе“Әдәӣж–Үд»¶пјҲеҗ«и·Ҝеҫ„пјүгҖҒйӘҢиҜҒз»“жһңгҖӮ
6. **иҮӘеҢ…еҗ«** вҖ” дёҚдҫқиө–"д№ӢеүҚеҜ№иҜқ"гҖӮжүҖжңүйңҖиҰҒзҡ„дҝЎжҒҜйғҪеңЁжҢҮд»Өе’ҢйЎ№зӣ®ж–Үд»¶дёӯгҖӮ

export async function POST(req: Request) {
  const { mode, messages, professorContext, userStyleProfile } = await req.json();
  
  // 1. еҠ иҪҪеҜ№еә”жЁЎејҸзҡ„ System Prompt
  let systemPrompt = getBaseSystemPrompt();  // е…ЁеұҖе“ҒзүҢзәўзәҝ
  systemPrompt += getModePrompt(mode);       // жЁЎејҸдё“з”ЁжҢҮд»Ө
  
  // 2. еҰӮжһңжңүз”ЁжҲ·йЈҺж јз”»еғҸпјҢиҝҪеҠ еҲ° System Prompt
  if (userStyleProfile) {
    systemPrompt += `\n\nз”ЁжҲ·иҜҙиҜқйЈҺж јпјҡ${describeStyle(userStyleProfile)}гҖӮиҜ·еҢ№й…Қз”ЁжҲ·йЈҺж јеӣһеӨҚгҖӮ`;
  }
  
  // 3. еҰӮжһңжҳҜз§‘з ”ж·ұжҪңжЁЎејҸпјҢжү§иЎҢ RAG
  let ragContext = "";
  if (mode === "research") {
    const lastMsg = messages[messages.length - 1].content;
    
    // е№¶иЎҢжЈҖзҙўпјҲPromise.allпјҢдёҚиҰҒдёІиЎҢпјҒпјү
    const [papers, knowledge, professors] = await Promise.all([
      searchSemanticScholar(lastMsg),      // lib/server/semantic-scholar.ts
      searchKnowledgeBase(lastMsg),         // lib/server/rag-engine.ts
      searchProfessorsByTag(lastMsg),       // lib/server/matching-engine.ts
    ]);
    
    ragContext = assembleRAGContext(papers, knowledge, professors);
    systemPrompt += `\n\nжЈҖзҙўеҲ°зҡ„еҸӮиҖғиө„ж–ҷпјҡ\n${ragContext}`;
  }
  
  // 4. еҰӮжһңжңүж•ҷжҺҲ contextпјҲд»ҺиҜҰжғ…йЎөи·іиҪ¬пјү
  if (professorContext) {
    systemPrompt += `\n\nз”ЁжҲ·жӯЈеңЁдәҶи§Јзҡ„ж•ҷжҺҲпјҡ${JSON.stringify(professorContext)}`;
  }
  
  // 5. и°ғз”Ё Claude API
  const response = await anthropic.messages.create({
    model: "claude-sonnet-4-6",
    max_tokens: 2000,
    system: systemPrompt,
    messages: messages.map(m => ({ role: m.role, content: m.content })),
  });
  
  // 6. еҗҺеӨ„зҗҶ
  const reply = response.content[0].text;
  const result = postProcess(reply, mode);  // жҸҗеҸ–еј•з”ЁгҖҒжЈҖжҹҘж— жқҘжәҗж•°жҚ®
  
  // 7. дҝқеӯҳеҜ№иҜқи®°еҪ•еҲ°ж•°жҚ®еә“
  await saveConversation(mode, messages, result);
  
  // 8. жЈҖжҹҘжҳҜеҗҰеә”иҜҘеј•еҜје’ЁиҜў
  result.suggestConsultation = shouldSuggestConsultation(messages, mode);
  
  // 9. жЈҖжҹҘжҳҜеҗҰи§ҰеҸ‘жҲҗе°ұ
  result.achievement = checkAchievements(messages, mode);
  
  return Response.json(result);
}

```sql
-- еҝ…йЎ»еңЁ Supabase SQL Editor дёӯеҲӣе»әжӯӨеҮҪж•°
CREATE OR REPLACE FUNCTION match_knowledge(
  query_embedding vector(1536),
  match_threshold float DEFAULT 0.7,
  match_count int DEFAULT 10
)
RETURNS TABLE (
  id uuid,
  source_type text,
  source_title text,
  content text,
  similarity float
)
LANGUAGE sql STABLE
AS $$
  SELECT id, source_type, source_title, content,
    1 - (embedding <=> query_embedding) AS similarity
  FROM knowledge_chunks
  WHERE 1 - (embedding <=> query_embedding) > match_threshold
  ORDER BY embedding <=> query_embedding
  LIMIT match_count;
$$;
```
