# 推广中心海报编辑器设计规格
> 版本: V1.0 | 日期: 2026-05-23 | 状态: 确定稿,不可随意覆盖

## 核心原则
图片海报 = 渐变海报的克隆体。两者使用同一套编辑器逻辑、同一套 UI 布局、
同一套预设元素、同一套编辑能力。唯一区别是背景(渐变色 vs 图片)。
Canvas 尺寸固定,图片适应 canvas,不是 canvas 适应图片。

## Tab 结构
推广链接 | 推广二维码 | 渐变海报 | 图片海报

## Canvas 规格
- 固定画布: 3:4(1080x1440) / 1:1(1080x1080) / 9:16(1080x1920)
- 两个 tab 都支持 3 种尺寸切换
- 导出: canvas.toDataURL('image/png', 1.0)

## 预设元素(两 tab 一致,加载即有)
- Koala PhD logo: 左上角,锁定
- 主标题:「用 AI 找到你的理想 PhD 导师」白色 48px 粗体
- 副标题:「覆盖澳洲38所大学、24,000+位教授」白色 24px
- 4 条卖点(独立文字框 白色 20px):
  AI智能匹配/一键套磁信/研究计划对齐/申请进度追踪
- www.koalaphd.com 底部(不放邮箱)
- QR码: agent referral_code 动态生成,下半部居中
- 邀请码 + 渠道标识
- 深色背景白字;简约白底用深灰
- 所有元素可拖拽+可缩放+双击编辑

## 左侧面板(两 tab 一致)
- 文案输入(主标题/副标题)
- 渠道选择下拉
- 显示开关 checkbox(二维码/网址/邀请码/渠道)
- 尺寸切换 3:4/1:1/9:16
- 添加元素(+文字/+QR码)
- 重新生成 + 下载PNG

## 文字编辑
双击后浮动工具条:
- 字体 Google Fonts: 中文(思源黑体/思源宋体/站酷快乐体/阿里巴巴普惠体/霞鹜文楷)
  英文(Inter/Montserrat/Playfair Display/Poppins/Roboto)
- 字号滑杆、颜色(白/黑/#D4A843)、粗体
- FontFace API 预加载,document.fonts.ready 后初始化 canvas

## Tab 3 渐变海报
- 3 主题: 简约(白底)/学术(深蓝)/活力(橙粉),各有 A/B 版
- 蒙层: 顶部白→透明渐变 30%

## Tab 4 图片海报
- 6 张底图: /images/posters/11~66.png
- canvas 固定 1080x1440,backgroundImage scaleToWidth/Height 取大值居中
- 蒙层: #0F1419 50% + 顶底 scrim,不用 CSS blur

## 修改规则
改海报前必读本文档;不得删除已有功能;改完更新本文档。
